

## Overview

Minimal project demonstrating how to install and run Raspberry Pi OS on a virtual machine.

## Setup

* Download Raspberry Pi OS
* Create VM (Linux → Debian)
* Allocate RAM & CPU
* Attach image and start

## Requirements

Raspberry Pi OS image • VirtualBox • Basic Linux tools

## Key Points

* Lightweight, low-cost system
* Used for IoT, programming, and learning
* Supports virtualization environments ([thelinuxvault][1])

## System Concept

pivot_root() → changes root filesystem
